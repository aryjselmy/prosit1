# ZooManagement

Ce projet est une application Java développée avec IntelliJ IDEA pour gérer un zoo.

## Fonctionnalités
- Saisie du nom du zoo
- Saisie du nombre de cages
- Affichage des informations du zoo

## Technologies utilisées
- Java
- IntelliJ IDEA
- Git & GitHub
